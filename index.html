Lista de Etiquetas HTML

Lista de Etiquetas HTML
NO. Nombre de la Etiqueta Descripción Ejemplo de Uso Elemento Adicional
1 <a> Define un hipervínculo. Se utiliza para enlazar una página a otra o un recurso. <a href="https://www.example.com">Visita Ejemplo</a> Se puede agregar el atributo target="_blank" para abrir el enlace en una nueva pestaña.
2 <abbr> Define una abreviatura o acrónimo. <abbr title="Hypertext Markup Language">HTML</abbr> El atributo title proporciona el texto completo de la abreviatura.
3 <address> Define información de contacto. <address>123 Calle Principal, Ciudad</address> Usado para mostrar información de contacto del autor o propietario del documento.
4 <area> Define una área dentro de un mapa de imagen. <map name="mapa"><area shape="rect" coords="34,44,270,350" href="pagina.html"></map> Se usa junto con <map> y <img> para definir áreas clicables.
5 <article> Define un artículo independiente. <article><h2>Título</h2><p>Contenido</p></article> Usado para agrupar contenido que puede ser distribuido de forma independiente.
6 <aside> Define contenido adicional o secundario. <aside>Contenido relacionado</aside> Generalmente usado para contenido lateral o complementario.
7 <audio> Define un contenedor para contenido de audio. <audio controls><source src="audio.mp3" type="audio/mp3"></audio> Permite reproducir archivos de audio en la página.
8 <b> Define texto en negrita. <b>Texto en negrita</b> Generalmente se usa para resaltar texto sin importancia semántica.
9 <blockquote> Define una cita en bloque. <blockquote>Texto de cita</blockquote> Generalmente se muestra con una sangría.
10 <br> Define un salto de línea. Texto<br>Texto en la siguiente línea Usado para forzar un salto de línea en el texto.
11 <button> Define un botón clickable. <button>Hacer clic</button> Puede ser de tipo "submit", "button", o "reset".
12 <canvas> Define un área para gráficos dinámicos. <canvas id="miCanvas" width="200" height="100"></canvas> Usado para dibujar gráficos usando JavaScript.
13 <caption> Define un título para una tabla. <table><caption>Título de la tabla</caption></table> Se usa dentro del elemento <table>.
14 <cite> Define una referencia a una obra. <cite>Título del libro</cite> Generalmente se muestra en cursiva.
15 <code> Define un fragmento de código. <code>console.log('Hola Mundo');</code> Generalmente se muestra en una fuente monoespaciada.
16 <col> Define propiedades de una columna en una tabla. <col span="2"> Se usa dentro de <colgroup> para aplicar estilos a columnas específicas.
17 <colgroup> Define un grupo de columnas en una tabla. <colgroup><col span="2"></colgroup> Usado para aplicar estilos o atributos a un grupo de columnas.
18 <data> Define un valor asociado con un contenido. <data value="2024-09-10">10 de septiembre de 2024</data> Permite añadir datos semánticos a los elementos.
19 <datalist> Define una lista de opciones predefinidas para un campo de entrada. <input list="opciones"><datalist id="opciones"><option value="Opción 1"><option value="Opción 2"></datalist> Proporciona sugerencias para el campo de entrada.
20 <del> Define texto que ha sido eliminado. <del>Texto eliminado</del> Generalmente se muestra con una línea a través del texto.
21 <details> Define un control de visibilidad para contenido adicional. <details><summary>Resumen</summary><p>Contenido adicional</p></details> Permite mostrar u ocultar contenido adicional.
22 <dfn> Define un término que está siendo definido. <dfn>HTML</dfn> es un lenguaje de marcado. Generalmente se muestra en cursiva.
23 <dialog> Define un cuadro de diálogo o ventana modal. <dialog>Este es un cuadro de diálogo</dialog> Se usa para crear ventanas modales en la página.
24 <div> Define una sección o contenedor en el documento. <div>Contenido del div</div> Usualmente usado para aplicar estilos o agrupar otros elementos.
25 <dl> Define una lista de definiciones. <dl><dt>Término</dt><dd>Definición</dd></dl> Contiene <dt> para términos y <dd> para definiciones.
26 <dt> Define un término en una lista de definiciones. <dl><dt>Término</dt><dd>Definición</dd></dl> Usado dentro de <dl>.
27 <em> Define texto enfatizado. <em>Texto enfatizado</em> Generalmente se muestra en cursiva.
28 <embed> Define un contenedor para contenido externo. <embed src="video.mp4" type="video/mp4"> Permite incrustar contenido multimedia.
29 <fieldset> Define un grupo de controles en un formulario. <fieldset><legend>Información personal</legend></fieldset> Generalmente se usa para agrupar controles de formulario.
30 <figcaption> Define un pie de foto para un elemento <figure>. <figure><img src="imagen.jpg" alt="Descripción"><figcaption>Descripción de la imagen</figcaption></figure> Se usa dentro de <figure> para describir imágenes o gráficos.
31 <figure> Define contenido ilustrativo independiente. <figure><img src="imagen.jpg" alt="Descripción"><figcaption>Descripción</figcaption></figure> Usado para agrupar contenido visual y su descripción.
32 <footer> Define el pie de página de un documento o sección. <footer><p>Derechos reservados</p></footer> Generalmente contiene información de contacto o derechos de autor.
33 <form> Define un formulario para entrada de datos. <form action="/enviar"><input type="text"></form> Contiene elementos como <input>, <select>, y <textarea>.
34 <h1> Define el encabezado más importante en la página. <h1>Este es un Encabezado</h1> Los encabezados van desde <h1> (más importante) hasta <h6> (menos importante).
35 <h2> Define el segundo encabezado más importante en la página. <h2>Encabezado nivel 2</h2> Usado para subencabezados de <h1>.
36 <h3> Define el tercer encabezado más importante en la página. <h3>Encabezado nivel 3</h3> Usado para subencabezados de <h2>.
37 <h4> Define el cuarto encabezado más importante en la página. <h4>Encabezado nivel 4</h4> Usado para subencabezados de <h3>.
38 <h5> Define el quinto encabezado más importante en la página. <h5>Encabezado nivel 5</h5> Usado para subencabezados de <h4>.
39 <h6> Define el sexto encabezado más importante en la página. <h6>Encabezado nivel 6</h6> Usado para subencabezados de <h5>.
40 <head> Define la sección de encabezado del documento. <head><title>Título</title></head> Contiene metadatos, enlaces a estilos, y scripts.
41 <header> Define el encabezado de un documento o sección. <header><h1>Título</h1></header> Generalmente contiene elementos de navegación y encabezados.
42 <hgroup> Define un grupo de encabezados. <hgroup><h1>Título</h1><h2>Subtítulo</h2></hgroup> Se usa para agrupar varios niveles de encabezado.
43 <hr> Define una línea horizontal. <hr> Se usa para separar contenido temáticamente.
44 <html> Define el documento HTML. <html><head></head><body></body></html> El elemento raíz que contiene todo el contenido del documento.
45 <i> Define texto en cursiva. <i>Texto en cursiva</i> Generalmente se usa para texto que requiere énfasis visual.
46 <iframe> Define un marco embebido. <iframe src="pagina.html"></iframe> Permite incrustar otro documento HTML dentro de la página actual.
47 <img> Incrusta una imagen en la página. <img src="imagen.jpg" alt="Descripción de la imagen"> Importante incluir el atributo alt para accesibilidad.
48 <input> Define un campo de entrada. <input type="text" name="nombre"> El atributo type define el tipo de entrada (text, password, email, etc.).
49 <ins> Define texto que ha sido insertado. <ins>Texto insertado</ins> Generalmente se muestra con una línea debajo del texto.
50 <kbd> Define entrada del teclado. <kbd>Ctrl+V</kbd> Generalmente se muestra en una fuente monoespaciada.
51 <label> Define una etiqueta para un elemento de formulario. <label for="nombre">Nombre:</label><input type="text" id="nombre"> El atributo for asocia la etiqueta con un campo de entrada.
52 <legend> Define una leyenda para un grupo de controles en un formulario. <fieldset><legend>Información personal</legend></fieldset> Usado dentro de <fieldset> para describir el grupo de controles.
53 <li> Define un elemento de una lista. <ul><li>Elemento de lista</li></ul> Usado dentro de <ul> (lista desordenada) o <ol> (lista ordenada).
54 <link> Define la relación entre el documento actual y un recurso externo. <link rel="stylesheet" href="estilo.css"> Usado para vincular hojas de estilo externas.
55 <main> Define el contenido principal del documento. <main>Contenido principal</main> Debería ser único en la página y contener el contenido principal.
56 <map> Define un mapa de imagen. <map name="mapa"><area shape="rect" coords="34,44,270,350" href="pagina.html"></map> Se usa con <img> para definir áreas clicables dentro de una imagen.
57 <mark> Define texto marcado. <mark>Texto marcado</mark> Se muestra con un fondo amarillo.
58 <meta> Define metadatos sobre el documento. <meta charset="UTF-8"> Se usa para especificar información como el conjunto de caracteres o la descripción.
59 <meter> Define una medición en un rango. <meter value="0.6" min="0" max="1">60%</meter> Usado para representar valores en un rango definido.
60 <nav> Define un área de navegación. <nav><a href="#inicio">Inicio</a></nav> Generalmente contiene enlaces de navegación.
61 <noframes> Define contenido alternativo para navegadores que no soportan frames. <noframes>Tu navegador no soporta frames.</noframes> Se usa dentro de <frameset>.
62 <noscript> Define contenido alternativo para navegadores que no soportan JavaScript. <noscript>JavaScript está deshabilitado.</noscript> Se muestra si el navegador tiene JavaScript deshabilitado.
63 <object> Define un contenedor para contenido multimedia. <object data="video.mp4" type="video/mp4"></object> Se usa para incrustar diferentes tipos de contenido como vídeos o documentos.
64 <ol> Define una lista ordenada. <ol><li>Primer elemento</li><li>Segundo elemento</li></ol> Los elementos se enumeran automáticamente.
65 <optgroup> Define un grupo de opciones dentro de un elemento <select>. <select><optgroup label="Grupo 1"><option value="1">Opción 1</option></optgroup></select> Usado para agrupar opciones en un menú desplegable.
66 <option> Define una opción en un elemento <select>. <select><option value="1">Opción 1</option></select> Usado dentro de <select> para definir opciones.
67 <output> Define el resultado de un cálculo. <output name="resultado">0</output> Generalmente usado con elementos de formulario y scripting.
68 <p> Define un párrafo. <p>Este es un párrafo.</p> Los párrafos tienen un espacio arriba y abajo por defecto.
69 <pre> Define texto preformateado. <pre>Texto con espaciado y formato exacto</pre> Se muestra en una fuente monoespaciada.
70 <progress> Define el progreso de una tarea. <progress value="50" max="100">50%</progress> Se usa para mostrar el progreso en una barra de progreso.
71 <q> Define una cita corta en línea. <q>Texto de cita</q> Generalmente se muestra entre comillas.
72 <ruby> Define texto rubí. <ruby>漢字<rt>Kanji</rt></ruby> Se usa para mostrar pronunciaciones en textos en japonés y otros idiomas.
73 <s> Define texto que ya no es relevante o incorrecto. <s>Texto eliminado</s> Generalmente se muestra con una línea sobre el texto.
74 <samp> Define una muestra de salida del programa. <samp>Salida del programa</samp> Se muestra en una fuente monoespaciada.
75 <script> Define un script de programación. <script>console.log('Hola Mundo');</script> Generalmente se usa para agregar JavaScript a la página.
76 <section> Define una sección en el documento. <section><h2>Encabezado de sección</h2><p>Contenido</p></section> Se usa para dividir el documento en secciones temáticas.
77 <select> Define un menú desplegable. <select><option value="1">Opción 1</option></select> Contiene elementos <option> para seleccionar.
78 <small> Define texto más pequeño que el texto circundante. <small>Texto pequeño</small> Generalmente se muestra en un tamaño de fuente más pequeño.
79 <span> Define una sección en línea. <span>Texto en línea</span> Usado para aplicar estilos a una parte específica del texto.
80 <strong> Define texto importante o fuerte. <strong>Texto fuerte</strong> Generalmente se muestra en negrita.
81 <style> Define estilos CSS. <style>body { font-family: Arial; }</style> Se usa para agregar estilos CSS directamente al documento HTML.
82 <sub> Define texto en subíndice. <sub>Texto en subíndice</sub> Se usa para notaciones químicas y matemáticas.
83 <summary> Define un resumen para un elemento <details>. <details><summary>Resumen</summary><p>Contenido</p></details> Se muestra como el encabezado del contenido colapsable.
84 <sup> Define texto en superíndice. <sup>Texto en superíndice</sup> Se usa para notaciones matemáticas y químicas.
85 <table> Define una tabla. <table><tr><td>Celda</td></tr></table> Contiene elementos como <tr> (filas) y <td> (celdas).
86 <tbody> Define el cuerpo de una tabla. <table><thead><tr><th>Encabezado</th></tr></thead><tbody><tr><td>Celda</td></tr></tbody></table> Usado para agrupar filas en el cuerpo de una tabla.
87 <td> Define una celda en una tabla. <table><tr><td>Celda</td></tr></table> Usado dentro de <tr> para definir una celda de datos.
88 <textarea> Define un área de texto multilínea. <textarea>Texto aquí</textarea> Permite a los usuarios ingresar varias líneas de texto.
89 <tfoot> Define el pie de una tabla. <table><tfoot><tr><td>Pie de tabla</td></tr></tfoot></table> Usado para agrupar el contenido del pie de tabla.
90 <th> Define una celda de encabezado en una tabla. <table><tr><th>Encabezado</th></tr></table> Usado dentro de <tr> para definir una celda de encabezado.
91 <thead> Define el encabezado de una tabla. <table><thead><tr><th>Encabezado</th></tr></thead><tbody><tr><td>Celda</td></tr></tbody></table> Usado para agrupar el encabezado de una tabla.
92 <time> Define una fecha/hora. <time datetime="2024-09-10">10 de septiembre de 2024</time> Usado para representar fechas y horas en un formato legible para las máquinas.
93 <title> Define el título del documento. <title>Título de la página</title> Se muestra en la barra de título del navegador.
94 <tr> Define una fila en una tabla. <table><tr><td>Celda</td></tr></table> Contiene celdas (<td> y <th>).
95 <u> Define texto subrayado. <u>Texto subrayado</u> Se usa para resaltar texto, aunque puede tener significados semánticos específicos.
96 <ul> Define una lista desordenada. <ul><li>Elemento de lista</li></ul> Los elementos se muestran con viñetas.
97 <var> Define una variable en un script. <var>Variable</var> Generalmente se usa para marcar variables en ejemplos de código.
98 <video> Define un contenedor para un vídeo. <video src="video.mp4" controls></video> Permite la reproducción de vídeos directamente en la página.
99 <wbr> Define una posible línea de quiebre en una palabra. <wbr> Indica al navegador dónde puede dividir la palabra si es necesario.
100 <address> Define información de contacto. <address>Contacto: ejemplo@dominio.com</address> Usado generalmente para información de contacto del autor o propietario del documento.
101 <details> Define un control de detalles. <details><summary>Resumen</summary>Contenido</details> Se usa para mostrar información adicional que se puede expandir y colapsar.
102 <dialog> Define un cuadro de diálogo o ventana modal. <dialog>Este es un cuadro de diálogo.</dialog> Puede ser mostrado usando JavaScript.
